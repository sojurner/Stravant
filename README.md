# Stravant

Instructions ---

clone down this rpo
```
git clone https://github.com/sojurner/Stravant.git
```
Run in terminal
```
npm install
```
```
node.serverjs
```
cd into Stravant-client

Run in terminal
```
npm install
```
```
npm start
```
